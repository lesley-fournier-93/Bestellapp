# 🍔 Food Ordering App

A food ordering web application inspired by Lieferando, built from scratch using HTML, CSS, and JavaScript.

## 🚀 Features

* Browse a restaurant menu with categorized dishes
* Add and remove items from the shopping cart
* Adjust item quantities dynamically
* Choose between pickup and delivery
* Automatic price calculation (including delivery costs)
* Simple checkout process with validation

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

* `index.html` – Main structure of the application 
* `style.css` – Styling and responsive design 
* `script.js` – Application logic (cart, ordering, interactions) 

## 📚 What I Learned

* Building a dynamic user interface with JavaScript
* Managing application state (shopping cart logic)
* Implementing real-world features like ordering and delivery options
* Structuring code into reusable functions
* Handling user input and validation
